# Protogenesis V5 — Prototype 0.3

Esta iteración agrega progresión real sobre la base 0.2.

## Nuevos sistemas

- `V5GeneSystem`: árbol génico de 4 anillos.
- `V5ApexSystem`: invocación de forma apex/premium.
- `V5WorldEventSystem`: eventos ambientales periódicos.
- Integración HUD: panel de genes con tecla `G` y apex con tecla `P`.
- Integración de genes con división, herencia, colonización y reciclaje.

## Controles nuevos

- `G`: abrir/cerrar panel de genes.
- `P`: intentar invocar la forma apex recomendada.

## Flujo recomendado de prueba

1. Crear escena desde `Protogenesis > V5 > Create Full Prototype Scene`.
2. Play.
3. Farmear recursos.
4. Abrir panel con `G`.
5. Activar un gen de Anillo 1 para definir dominio.
6. Activar Anillo 2 para rol funcional.
7. Dividir y colonizar.
8. Llegar a Anillo 3 y 4.
9. Invocar apex con `P`.
