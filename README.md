# Proyecto Protogenesis IA

Prototipo de estrategia y simulacion biologica desarrollado en Unity.

## Requisitos

- Unity 6.3 LTS (`6000.3.9f1`)
- Git

## Abrir el proyecto

1. Clonar este repositorio.
2. Abrir la carpeta raiz desde Unity Hub.
3. Esperar a que Unity regenere `Library/` e importe los paquetes.
4. Abrir `Assets/_ProjectV5/Scenes/V5_Core.unity`.

## Contenido versionado

- `Assets/`: codigo, escenas y recursos del juego.
- `Packages/`: manifiesto y versiones de paquetes.
- `ProjectSettings/`: configuracion del proyecto Unity.

Las carpetas generadas por Unity, builds locales, logs, capturas y backups no se incluyen en Git.
