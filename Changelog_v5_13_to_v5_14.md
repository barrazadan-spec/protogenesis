# Changelog v5.13 -> v5.14

## Cambio Central

Se pivota el sistema biologico post-MVP hacia un **Sistema Canonico de Adaptaciones**.

Antes:

```text
Genes + Estructuras + Rutas + Affinity Tracker
```

Ahora:

```text
Adaptaciones como fuente de verdad
Identidades emergentes reconocidas por combinacion
Produccion y cuerpo derivados de esas adaptaciones
```

## Decisiones Agregadas

- Se separa **Adesina basica** de **Adhesion Persistente**.
- Los **hitos evolutivos** no cuentan contra el cap de 12 adaptaciones activas.
- El ambiente otorga descuentos/aceleradores, no bloqueos absolutos.
- Solo Nucleo y Adhesion Persistente son irreversibles canonicas por ahora.
- Tardigrado queda como estado/neutral/apex raro, no como ruta core.
- El panel G pasa a ser **Panel del Genoma / Adaptaciones**.
- El arbol visual gigante queda relegado a Codex o vista avanzada.
- La migracion tecnica debe ser incremental sobre V5, no reescritura V7 inmediata.
